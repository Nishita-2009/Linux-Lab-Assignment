Question 4 - File Access and I/O Investigation

1. Open Files
Open files were identified using the lsof command.

2. File Descriptor Observation
Linux uses file descriptors to manage file access. Standard input, output, and error are represented by descriptors 0, 1, and 2.

3. Output Redirection
The output of the ls command was redirected successfully to output.txt.

4. Error Redirection
The error generated by accessing a nonexistent file was redirected to error.txt.

5. Resource Limits
System resource limits were displayed using the ulimit command.

6. Conclusion
Linux efficiently manages file input/output through file descriptors and supports output/error redirection for better process management.
